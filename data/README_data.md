# Data

The original BRFSS 2023 data file was downloaded from the Centers for Disease Control and Prevention:

https://www.cdc.gov/brfss/annual_data/annual_2023.html

The raw data file is not included in this repository due to file size. To reproduce the analysis, download the 2023 SAS XPORT file from the CDC and place it in the `data/` folder.

Expected file name:

`LLCP2023.XPT`
